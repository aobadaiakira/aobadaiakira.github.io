カクヨム形式ファイル・ビューワーを作った

前々から欲しいと思っていた、カクヨム形式のファイル・ビューワを作った。

検索すれば、Visual Studio Code をはじめとする有名テキスト・エディタ用プラグインなどが見つかるのかも知れないが、やはりエディタはエディタ、ビューワはビューワで別に使った方が利便性が高いと思って、自作してみた。
エディタとビューワを別々のアプリにしておけば、ブラグインの有る無しに関わらず好きなエディタを選べる。
両者の結合度を低くしておきたい、と言えば良いか。

まあまあ便利なアプリが出来たと思う。
汎用性は無いが、どうせ私が自分で使うだけだ。
私にとって便利であれば、それで良い。

他人様に見せられるような代物では無いが、気が向いたら github に上げるかも知れない。
